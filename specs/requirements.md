# Functional Requirements

- `REQ-FUNC-001`: O sistema deve interpretar solicitações em linguagem natural.
- `REQ-FUNC-002`: O `agent-service` deve planejar ações com base em contexto recuperado.
- `REQ-FUNC-003`: Toda ação executável deve passar pelo `mcp-service`.
- `REQ-FUNC-004`: O ERP legado deve ser acessado exclusivamente via `erp-acl-service`.
- `REQ-FUNC-005`: Toda execução de ação deve gerar registro auditável.
